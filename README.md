# blippy

## Наш проект
X-Baпк — оффлайн P2P-платёжная система: переводы валюты DOD напрямую между ноутами по Bluetooth без интернета.
Есть кошельки, транзакции, история операций.

# Предполагаемое разделение задач

## Алексей Синицын - Core + бизнес-логика
### Основное задание
• wallet/
    • класс Wallet (баланс, операции)
• transaction/
• модель (pydantic)
• логика:
    • создание транзакции
    • валидация
    • применение
• обработка ошибок
### Фичи
• создание кошелька
• отправка/получение DOD
• проверка баланса
• история транзакций


## Борис Осин - Bluetooth / Network
### Основное задание
• network/
  • отправка транзакций по Bluetooth (bleak)
  • приём транзакций
  • сериализация (JSON)
• простой протокол:
  • отправка
  • подтверждение получения
### Фичи:
• P2P переводы между ноутами
• обработка ошибок соединения
• ретраи (повтор отправки)


## Матвей Вербицкий - Storage / БД
### Основное задание
• storage/
  • SQLite (sqlite3)
  • таблицы:
    • wallet (id, balance)
    • transactions (id, sender, receiver, amount, timestamp)
• repository слой
### Фичи
• сохранение баланса
• сохранение всех транзакций
• загрузка состояния при старте
• базовая защита от повторных транзакций (по id)
