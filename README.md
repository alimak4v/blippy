# X-Bank: Оффлайн P2P-платёжная система

## Основная идея
**X-Bank** - оффлайн P2P-платёжная система для прямых переводов валюты `DOD` между ноутбуками по Bluetooth без доступа к интернету. 
Система включает в себя локальные кошельки, механизм транзакций, историю операций и надёжное хранение данных.
